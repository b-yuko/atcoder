# atcoder

AtCoder プログラミング用リポジトリです。  
Kotlin・TypeScript で問題を解くためのテンプレートやサンプル実装、テストコードを管理しています。

## セットアップ・ビルド方法

### Kotlin
- 必要: Java 11以上, Gradle
- ビルド: `cd kotlin-code && ./gradlew build`
- テスト: `cd kotlin-code && ./gradlew test`

### TypeScript
- 必要: Node.js, npm
- 依存インストール: `cd typescript-code && npm install`
- テスト: `cd typescript-code && npm test`

## テンプレート・サンプル
- Kotlin: `src/main/kotlin/template/` にテンプレートコードあり
- TypeScript: `src/main/template/` にテンプレートコードあり
- `abs/` ディレクトリにAtCoderの練習問題サンプル実装あり
